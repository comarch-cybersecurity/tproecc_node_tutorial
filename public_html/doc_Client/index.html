tProEccAPI.js

Browser side communication with tPro Ecc token is being handled by TProEccAPI class, which provides interface to directly access device functionalities such as:
- token initialization/deinitialization,
- generation of new ECC key-pair as well as their erasure,
- digital signature of delivered content,
- functions concerning PIN policy,
- obtaining detailed information about token's state.
